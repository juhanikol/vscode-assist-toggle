# Quick start

```bash
git clone github.com/juhanikol/vscode-assist-toggle.git
cd vscode-assist-toggle
./install.sh
```

Open a new terminal after installation, or run:

```bash
source ~/.bashrc
vassist --help
```

## Use from any project

Run `vassist` from the root of the project you want to change:

```bash
cd /path/to/project
vassist
```

See [README.md](README.md) for installation, dependencies, modes, restore, and uninstall instructions.

After installing, run `vassist doctor` from a project root to check the setup without changing settings.
